# Wismut Game Engine

Welcome to the repository of Wismut, a compact and educational-focused game engine, primarily developed in C++. This engine is a personal project, aimed at self-education and exploration in the realm of game engine architecture. Though modest in size, Wismut is designed to be a powerful tool for those who are also interested in learning about the inner workings of game development technologies.

## Features

- **Language**: Entirely written in C++ for performance and low-level control.
- **Multi-Platform Support**: Designed to be cross-platform, Wismut aims to support various operating systems.
- **Graphics API**: Currently supports Vulkan, with plans to include more APIs in the future. This approach allows for high efficiency and control over graphics rendering.
- **Educational Focus**: Developed with the intent of exploring and understanding the intricacies of game engine architecture.
